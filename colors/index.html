Colors

They were the first thing to be documented internally in an attempt to achieve a more consistent Design Language for WordPress.com. Colors are one of the most notable aspects in brand recognition.

WordPress.com Blue

The main color. Derived originally from the core WordPress blues, but evolved into a much brighter and saturated color.

  • WordPress.com Blue
  • #0087be
  • rgb(0,135,190)
  • $wordpress-blue

Secondary Blues

These complementary blues provide a set of secondary options for elements where the primary blue (or it's derivates) isn't appropriate. A good example is a large color block, like a background, might work better with a Medium Blue, than with the Primary WordPress.com blue.

  • Light Blue
  • #78dcfa
  • rgb(120,220,250)
  • $light-blue
  • Medium Blue
  • #00aadc
  • rgba(0,170,220)
  • $medium-blue
  • Dark Blue
  • #005082
  • rgb(0,80,130)
  • $dark-blue
