Typography

Expressive and utilitarian, serif and sans, our typefaces serve distinct roles: Merriweather is for content, Open Sans is for interface.

Typefaces

WordPress.com uses two typefaces: Merriweather and Open Sans. Both are released under Open Source licenses and are served via Google Fonts.

Merriweather

Designed by Eben Sorkin, Merriweather is a graceful, modern serif typeface. We use Merriweather only for user content. This includes posts and post titles, comments, pages; any text that a user creates.

Open Sans

Open Sans is a humanist sans serif typeface designed by Steve Matteson. The tone of Open Sans is utilitarian, and as such, we use Open Sans exclusively for text that’s part of our UI: headers, form labels, buttons, any text that’s not the user’s content.

Weights and Styles

We use the light (300) and bold (700) weights of Merriweather. Use the light (300) weight anywhere you’d usually use normal (400) weight text.

We use the light (300), regular (400), and semibold (600) weights of Open Sans. Anywhere you’d ordinarily use a bold (700) weight, use semibold (600) instead.

When specifying a font-weight in CSS, always use the numeric value instead of keywords ("400" rather than "normal") to ensure that the correct weights are being used. Always use the browser-default antialiasing; don’t disable subpixel antialiasing using the CSS font-smoothing property.

Typographic Modular Scale

A harmonic ratio helps in creating a more harmonious design. If we use the same scale across WordPress.com, things feel more cohesive — it’s as much about consistency as it is about harmony. Instead of using arbitrary numbers, we conform to a harmonic scale.

We use a double-stranded Perfect Fifth scale, based on the ideal text size of 16px and a secondary important number of 14px. We round the values to the nearest pixel for sanity. That gives us the following scale:

Pixels Ems
54 3.375
47 2.953
36 2.25
32 1.969
24 1.5
21 1.313
16 1
14 0.875
11 0.667
